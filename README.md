# dream-test-task

test task for my "dream" job

```
* Задача:

Разработать сервис для сокращения URL-адресов, который позволяет пользователям создавать короткие ссылки, отслеживать их использование и управлять ими. Сервис должен быть реализован на языке программирования Go.

Организация архитектуры и структуры кода будет учитываться в рамках оценки выполненного задания.

Выполненный код должен быть оформлен и предоставлен ссылкой на Github репозиторий.

* Требования:

[1] POST: /shortener

{
  "url": ""
}

- Пользователь вводит длинный URL в формате JSON и система генерирует короткий уникальный идентификатор для ссылки в ответ.
- Важно защитить систему от создания дублирующихся коротких ссылок.
- Реализовать ограничение по времени жизни коротких ссылок (например, 30 дней).

[2] GET: /shortener

- Пользователь должен иметь возможность просматривать список своих созданных ссылок.

[3] GET: /{link}

- При обращении к короткой ссылке пользователь должен быть перенаправлен на соответствующий длинный URL.

[4] DELETE: /{link}

- Пользователю должен быть доступен функционал для удаления своей короткой ссылки.

[5] GET: /stats/{link}

- Пользователю должна быть доступна статистика по учету коль-во переходов по своим коротким ссылкам, дата последнего перехода.


- Реализуйте работу с базой данных.
- Реализуйте простую валидацию длинных URL (например, проверка на корректность формата, пустые значения).
- Обеспечьте обработку ошибок (например, сокращённая ссылка не найдена).
- Подготовьте функциональное тестирование разработанного кода.

* Дополнительно:

- Реализовать Swagger документацию.
```
